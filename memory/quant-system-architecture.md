# 量化交易系统核心架构

> 学习笔记：四大核心模块

---

## 一、Data Ingestion（数据摄取）

摄取并处理结构化与非结构化数据。

### 1. 价格数据处理
- **前复权/后复权自动处理**
  - 前复权：以当前价格为基准调整历史价格
  - 后复权：以上市价格为基准调整历史价格
  - 分红、配股、拆股等除权除息处理

### 2. 股票筛选
- **识别并剔除ST/*ST股票**
  - ST股票：连续两年亏损
  - *ST股票：连续三年亏损，退市风险
  - 避免踩雷，降低风险

### 3. 特色数据更新
- **每日自动更新**
  - 龙虎榜数据（游资动向）
  - 北向资金（外资流向）
  - 融资融券数据
  - 大宗交易数据
  - 机构调研数据

---

## 二、Alpha Generation（多因子挖掘与合成）

提取产生超额收益的信号源（量价、基本面、情绪等）。

### 1. A股特色因子库
| 因子类型 | 具体因子 | 说明 |
|---------|---------|------|
| **量价因子** | 微盘股因子 | 小市值股票超额收益 |
| | 动量因子 | 近期涨幅强者恒强 |
| | 反转因子 | 超跌反弹 |
| | 波动率因子 | 高波动/低波动策略 |
| **基本面因子** | 价值因子 | PE/PB/PS低估值 |
| | 质量因子 | ROE/ROA/毛利率 |
| | 成长因子 | 营收增速/利润增速 |
| | 分红因子 | 股息率/分红稳定性 |
| **情绪因子** | 涨停因子 | 涨停家数/连板高度 |
| | 资金流向 | 主力净流入/散户净流出 |
| | 舆情因子 | 新闻情绪/社交媒体热度 |

### 2. 正交化处理
- **剔除共线性干扰**
  - 行业中性化：剔除行业影响
  - 市值中性化：剔除市值影响
  - 风格中性化：剔除风格因子影响
- **目的**：提取纯粹的Alpha信号

---

## 三、Backtesting Engine（高仿真事件驱动回测）

基于历史时间轴步进，验证策略的有效性。

### 1. T+1交易逻辑
- **强制T+1限制**
  - 当日买入，次日才能卖出
  - 真实模拟A股交易规则

### 2. 真实成交模拟
- **涨跌停板限制**
  - 涨停无法买入
  - 跌停无法卖出
- **停牌处理**
  - 停牌期间无法交易
  - 复牌后按规则处理

### 3. 成本计算
- **滑点（Slippage）**
  - 冲击成本：大单对价格的冲击
  - 延迟成本：下单到成交的时间差
- **交易成本**
  - 印花税：卖出时千分之一
  - 佣金：万分之2.5（双向）
  - 过户费、经手费等

---

## 四、Risk Control（动态风险剥离与归因）

测量暴露敞口，控制最大回撤。

### 1. 中性化控制
- **行业中性化**
  - 控制行业偏离度
  - 避免单一行业过度暴露
- **风格中性化**
  - 控制市值暴露
  - 控制价值/成长风格暴露

### 2. 系统性风险防护
- **硬性熔断机制**
  - 千股跌停时自动降仓
  - 大盘暴跌时止损离场
- **最大回撤控制**
  - 设定回撤阈值（如-10%）
  - 触及阈值强制清仓或降仓

### 3. 风险归因
- **收益归因**
  - 行业贡献
  - 风格贡献
  - 个股贡献
- **风险归因**
  - 系统性风险暴露
  - 非系统性风险暴露
