# A股交易系统

## 核心原则

### 分析基础
- 摒弃主观预测
- 仅基于资金行为与赔率进行概率推演

---

## 四层分析框架

### 1. 市场阶段（宏观温湿度）
- 冰点 / 启动 / 高潮 / 退潮
- 关注赚钱/亏钱效应极值

### 2. 板块地位（逻辑持续性）
- 主线 / 支线 / 轮动
- 判断依据：政策驱动、业绩驱动、资金驱动

### 3. 个股地位（辨识度）
- 龙头 / 跟风 / 补涨
- 关注筹码结构与获利盘比例

### 4. 资金行为（量价关系）
- 突破放量 / 调整缩量
- 警惕高位异常放量

---

## 三重风控标准（强制输出）

### 1. 胜率/赔率预估
- 高/中/低评级
- 核心逻辑支撑
- 无明确数据时，置信度设为低

### 2. 动态仓位建议
- 基于当前情绪
- 最高暴露敞口不超过总资金X%
- 单只不超过X%

### 3. 机械化防守线
- 明确止损位：跌破某根均线或关键支撑位
- **无条件止损纪律**

---

## 交易风格

### 特征
- 短中期交易
- 情绪周期 + 龙头股 + 板块轮动
- 严格止损

### 持仓管理
- 重仓股：航天发展、雷科防务、顺灏股份、利欧股份、天际股份
