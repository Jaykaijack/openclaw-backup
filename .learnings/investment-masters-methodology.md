# 投资大师智慧 - 核心方法论

## 一、巴菲特 & 芒格 - 价值投资

### 核心理念
1. **持续价值投资** - 60年6万倍收益的秘密
   - 平均每年约20%收益率
   - 关键在于"持续"而非"高收益"

2. **投资哲学演变**
   - 早期：格雷厄姆"捡烟蒂"法（买便宜货）
   - 后期：芒格影响下，以合理价格买入优质企业

3. **芒格的投资哲学**
   - 简单、长期、专注、理性
   - 投资不是复杂的艺术，而是简单的科学
   - 专注于自己擅长的领域

### 核心原则
- **能力圈原则**：只投资自己能理解的行业和企业
- **安全边际**：以显著低于内在价值的价格买入
- **长期持有**：陪伴优质企业共同成长
- **逆向思维**：在市场恐慌时贪婪

## 二、彼得·林奇 - PEG选股法

### PEG指标公式
```
PEG = PE / (盈利增长率 × 100)
```

### 判断标准
| PEG值 | 含义 |
|-------|------|
| < 0.5 | 严重低估，值得投资 |
| 0.5-1.0 | 低估，有投资价值 |
| = 1.0 | 估值合理 |
| > 1.0 | 高估，需谨慎 |
| > 2.0 | 严重高估 |

### 适用条件
- 适用于盈利稳定的成长型企业
- 不适用于周期性企业
- 最佳增速区间：15%-30%

### 林奇选股要点
- 关注企业业务类型和管理能力
- 优先选择增长速度高的公司
- 在同样增长速度下选择市盈率较低的公司

## 三、杰西·利弗莫尔 - 关键点交易法

### 核心逻辑
- **市场支点**：判断趋势与进场的关键依据
- **关键点**：多空力量博弈的胜负分水岭

### 操作特征
1. **数字化与公式化**：每个步骤转化为数学公式
2. **行业视角判断趋势**：通过行业平均走势判断
3. **过滤价格噪声**：设置预设水平，避免被震荡出局
4. **跟踪领涨股票**：每个周期都有不同领涨股

### 交易准则
1. **趋势为王**：只做大趋势，顺势而为
2. **严格止损**：单笔损失不超过总资本1%-2%
3. **让利润奔跑**：盈利时不急于平仓
4. **情绪管理**：克制贪婪与恐惧
5. **耐心等待**：仅在"高胜率机会"出现时行动

### 加仓法则（金字塔加仓）
- 第一笔：20%资金试探
- 确认趋势后：加15%
- 趋势越强：加仓比例越小（最后可能只加5%）
- **铁律**：永不摊平亏损，只对盈利头寸加仓

## 四、威廉·欧奈尔 - CANSLIM法则

### CANSLIM七步选股

| 字母 | 含义 | 标准 |
|------|------|------|
| C | 当季每股盈余 | 同比增长20%以上，最好40%-100% |
| A | 年度收益增长率 | 连续3年每年增长25%以上，ROE≥17% |
| N | 新产品/新管理层/股价新高 | 至少满足一点 |
| S | 供给与需求 | 流通市值50-200亿，有股东回购 |
| L | 龙头股 | RPS≥80，行业前三 |
| I | 机构认可度 | 机构持股比例上升 |
| M | 市场方向 | 大盘20日均线向上，成交量放大 |

### 杯柄形态（买入信号）

#### 形态特征
1. **杯部**：U型回调（不是V型！）
   - 深度：前期涨幅的12%-33%（理想20%-30%）
   - 时间：1-6个月（周线图最佳）

2. **柄部**：小幅向下倾斜整理
   - 深度：仅回调杯右侧上涨高度的1/3以内
   - 时间：1-4周
   - 量能：缩量

3. **突破**：放量突破柄部高点
   - 量能：较调整期均值高40%-50%

#### 交易策略
- **买入时机**：突破柄部高点且放量
- **止损设置**：柄部最低点下方3%-5%
- **止盈目标**：杯部高度从突破点向上复制

## 五、马克·米勒维尼 - SEPA策略

### 7项严格筛选标准
1. 剔除ST和次新股
2. 营收增长率 > 25%
3. 净利润增长率 > 30%，环比为正
4. 股价 > 50日均线 > 150日均线
5. 10日均量 > 120日均量（放量）
6. ROE > 15%
7. 三年净利润复合增长 > 20%

### VCP形态要点
- 价格波动逐步收窄
- 成交量萎缩
- 突破前夜蓄势

### 第二阶段特征
- 股价站上200日均线
- 较年内低点上涨25%以上
- 50日均线 > 150日均线 > 200日均线

## 六、综合应用框架

### 选股流程
1. **基本面筛选**（CANSLIM/SEPA）
   - 营收增长 > 25%
   - 净利润增长 > 30%
   - ROE > 15%

2. **估值判断**（PEG）
   - PEG < 1.0
   - 安全边际充足

3. **技术形态**（杯柄/VCP）
   - 形成标准形态
   - 突破关键点

4. **资金管理**（利弗莫尔）
   - 金字塔加仓
   - 严格止损

### 止损纪律
- 亏损7-8%强制离场
- 单笔风险不超过总资金1%
- 永不摊平亏损
