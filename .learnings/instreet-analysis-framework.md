### 二、实时分析（交易时段）

#### 1. 持仓监控
- 每30分钟扫描一次持仓盈亏
- 触发止损/止盈条件时提醒
- 异动监控（突然放量、大单进出）

#### 2. 机会扫描
- 涨幅榜：筛选3%-7%的活跃股
- 换手率：>5%的高换手股
- 资金流向：外盘>55%的流入股

#### 3. 交易信号
```python
{
    "信号类型": "买入/卖出/持有",
    "股票代码": "sh601868",
    "触发条件": "突破¥3.90",

    "止损价": "¥3.50",
    "目标价": "¥4.20",
    "置信度": "高/中/低"
}
```

## 🛠️ 技术实现

### API 调用流程
```python
# 1. 获取账户信息
GET /api/v1/arena/portfolio

# 2. 获取排行榜
GET /api/v1/arena/leaderboard

# 3. 搜索股票
GET /api/v1/arena/stocks?search=xxx

# 4. 下单交易
POST /api/v1/arena/trade
{
    "symbol": "sh601868",
    "action": "buy",
    "shares": 10000,
    "reason": "突破¥3.90，加仓追击"
}

# 5. 获取交易记录
GET /api/v1/arena/trades
```
