## 智能体 1：情绪雷达 Agent（09:00 盘前）

### 核心职责
市场情绪温度计 + 竞价信号过滤器

### 输入数据
- 隔夜美股/亚太走势
- 涨跌停家数、连板梯队
- 09:25 竞价数据
- 隔夜消息（S级/A级/B级分类）

### 输出格式
```
【情绪水位】冰点 / 修复期 / 高潮期 / 退潮期
【竞价红绿灯】
  🔴 放弃：竞价不及预期
  🟡 观望：平开或小幅低开
  🟢 关注：缩量回踩均线或强势高开带量
【今日策略】重仓出击 / 轻仓试错 / 防守看戏
```

### 关键因子
| 因子 | 权重 | 判断标准 |
|-----|------|---------|
| 涨停梯队高度 | 25% | >5板为高潮，<3板为冰点 |
| 连板晋级率 | 25% | >70%为强，<40%为弱 |
| 跌停家数 | 20% | >10家为风险信号 |
| 竞价量比 | 20% | >3为异常，需警惕 |
| 隔夜消息 | 10% | S级可改变情绪走向 |

## 智能体 1.5：情绪雷达 Agent（09:30 竞价分析）

### 核心职责
集合竞价最终确认 + 开盘前最后校准

### 为什么需要 09:30 报告？
09:00-09:30 之间可能发生：
- 隔夜消息发酵（政策、业绩、外围市场变化）
- 大资金最后竞价抢筹或出货
- 情绪突然逆转（从乐观到恐慌或反之）

### 输入数据（09:25-09:30）
- 集合竞价最终成交价、成交量
- 涨停股封单金额和数量
- 板块内个股高开/低开分布
- 异常竞价预警（天地板/地天板）

### 分析维度

#### 1. 最高连板股竞价诊断
| 指标 | 强势信号 | 弱势信号 |
|-----|---------|---------|
| 高开幅度 | >5% 且 <10%（有空间） | >10%（一致预期，易炸）或 <2%（不及预期） |
| 量比 | 2-5（资金关注） | >10（过度一致）或 <1（无人关注） |
| 封单 | 逐步增加 | 快速减少 |
| 竞价走势 | 稳步抬升 | 高开低走 |

#### 2. 板块竞价强度评分
```
板块竞价强度 = (涨停股数量 × 40%) + (高开率 × 30%) + (板块龙头强度 × 30%)

评分标准：
• 90-100分：🔥 极强（板块爆发）
• 70-89分：➖ 中性偏强（有预期）
• 50-69分：➖ 中性（观望）
• <50分：❄️ 弱势（回避）
```

#### 3. 竞价异常检测
```
⚠️ 风险信号：
• 天地板预警：竞价从涨停砸到绿盘
• 地天板机会：竞价从跌停拉到红盘（极少见）
• 巨量低开：前5分钟成交量 > 昨日全天20%
• 板块集体低开：板块内 >50% 个股低开
```

### 输出格式
```
🎯 二郎竞价分析 [日期] 09:30

【竞价情绪】冰点/修复/高潮/退潮

【核心股竞价】
• 最高板XXX：高开X%，量比X，封单X万手 [超预期/符合/不及]
• 中军XXX：高开X%，量比X [强势/中性/弱势]

【板块竞价强度】
• XXX板块：X只涨停，高开率X%，强度评分X分 [🔥强/➖中/❄️弱]
• XXX板块：...

【竞价红绿灯】
🟢 绿灯（关注）：
  • XXX - 逻辑
🟡 黄灯（观望）：
  • XXX - 逻辑
🔴 红灯（放弃）：
  • XXX - 逻辑

【风险预警】
⚠️ XXX股：巨量低开，注意风险
⚠️ XXX板块：集体低开，回避

【修正结论】
09:00预判：XXX
09:30修正：XXX
今日策略：重仓出击/轻仓试错/防守看戏

【开盘重点关注】
1. XXX - 买点/观察点
2. XXX - 买点/观察点
```

### 与 09:00 报告的对比
| 维度 | 09:00 盘前预热 | 09:30 竞价分析 |
|-----|---------------|---------------|
| 数据 | 隔夜信息+预期 | 真实竞价结果 |
| 准确性 | 70% | 90% |
| 作用 | 提前准备 | 最终确认 |
| 策略 | 初步方向 | 可执行指令 |

## 智能体 2：深度研究 Agent（选股阶段）

### 核心职责
个股基本面 + 消息面 + 产业链深度分析

### 分析维度（参考 ValueCell DeepResearch）

#### 2.1 基本面扫描
- 财务数据：ROE、毛利率、现金流
- 估值水平：PE/PB 分位数
- 机构持仓：基金/北向资金变化
- 分析师评级：目标价、盈利预测

#### 2.2 消息面过滤
```
S级（全市场共振）：政策利好、行业变革、重大订单
A级（板块驱动）：业绩超预期、技术突破、产能扩张
B级（个股事件）：减持、无关研报、日常公告 → 剔除
```

#### 2.3 产业链定位
- 上游/中游/下游位置
- 供需格局分析
- 价格传导能力
- 竞争格局（CR5、护城河）

#### 2.4 龙虎榜拆解（15:30后）
- 顶级游资：章盟主、方新侠、作手新一
- 机构席位：净买入/卖出
- 拉萨天团：散户信号（反向指标）

### 输出格式
```
【股票名称】XXX
【核心逻辑】一句话总结炒作逻辑
【基本面评分】X/10
【消息面评级】S/A/B
【产业链位置】上游/中游/下游
【龙虎榜】游资/机构动向
【风险点】XXX
【买点】缩量回踩5日线 / 突破前高确认
【防守位】跌破XX元不看
```

## 智能体 3：策略执行 Agent（交易时段）

### 核心职责
买卖点判断 + 仓位管理 + 风控纪律

### 11:30 午盘校准

#### 早盘剧本复盘
- 实际走势 vs 09:00 预判
- 真突破 vs 假拉升（诱多识别）

#### 主线强度确认
- 上午成交额 TOP3 板块
- 板块涨停潮情况
- 龙头封单坚决度
- 跷跷板效应检测

#### 持仓体检
```
⚠️ 量价背离警报：
  - 放天量不涨 → 主力出货
  - 冲高回落破均价线 → 弱势

📋 下午操作纪律：
  - 跟风杂毛 → 下午开盘找高点切掉
  - 核心龙头回踩均价线不破 → 【可低吸】
```

### 仓位管理规则
| 情绪水位 | 仓位上限 | 操作风格 |
|---------|---------|---------|
| 冰点期 | 30% | 只试错龙头 |
| 修复期 | 50% | 低吸核心 |
| 高潮期 | 70% | 去弱留强 |
| 退潮期 | 20% | 空仓或逆回购 |

### 买卖点纪律
```
买点：
  ✅ 缩量回踩5日线/10日线
  ✅ 突破前高且放量确认
  ✅ 竞价超预期 + 板块共振

卖点：
  ❌ 放量滞涨
  ❌ 跌破关键均线无反弹
  ❌ 板块退潮时跟风股
```

## 智能体 4：复盘学习 Agent（15:00 收盘后）

### 核心职责
预测验证 + 错误归因 + 因子迭代

### 15:00 收盘清算

#### 全天情绪清算
- 亏钱效应：最惨票特征（高位断板/中军破位）
- 退潮极值判断
- 赚钱效应：首板新概念、抱团老妖

#### 龙虎榜与资金拆解
- 顶级游资合力方向
- 主力资金流向
- 拉萨天团动向（散户情绪）

### 次日交易剧本
```
【主线锁定】明天唯一值得看的板块
【猎物锁定】1-2只核心龙头/中军
【绝对防守位】跌破不看
【观察买点】如：缩量回踩5日线
【仓位上限】X%
```

### 准确率追踪表
| 日期 | 预判 | 实际 | 结果 | 错误原因 |
|-----|------|------|------|---------|
| 示例 | 修复期 | 退潮期 | ❌ | 忽视了XX利空 |

### 因子权重迭代（每周复盘）
```
当前权重配置（示例）：
- 情绪因子：40%
- 基本面因子：25%
- 资金流向：20%
- 消息面：15%

淘汰标准：连续2周负贡献
引入标准：回测胜率>60%
```
