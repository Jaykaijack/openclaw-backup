# 投研核心技能 - 深度学习笔记

## 一、行为金融学与投资心理

### 1. 常见心理偏差

| 偏差类型 | 表现 | 应对策略 |
|---------|------|---------|
| **羊群效应** | 盲目跟风买入/卖出 | 独立思考，逆向思维 |
| **损失厌恶** | 亏损1份需盈利2.5份才能平衡 | 基于资产本身决策，忽略浮盈浮亏 |
| **锚定效应** | 把买入价当心理锚点 | 关注企业价值，而非成本价 |
| **确认偏误** | 只看支持自己观点的信息 | 主动寻找反面证据 |
| **过度自信** | 高估自己的投资能力 | 保持谦逊，承认市场不可预测 |
| **时间贴现** | 更看重眼前收益 | 关注长期价值，耐心持有 |

### 2. 牛熊市心理特征
- **牛市**：浮盈时担忧回撤，急于落袋为安 → 拿不住
- **熊市**：浮亏时不肯认输，死扛到底 → 套得牢

### 3. 克服心理偏差的方法
1. 建立交易系统，用规则对抗情绪
2. 每笔交易独立看待，不摊平亏损
3. 关注资产质量和未来前景，而非浮盈浮亏
4. 保持逆向思维习惯，利用均值回归规律

## 二、宏观经济分析框架

### 1. 宏观政策四大目标
- 经济增长
- 充分就业
- 物价稳定
- 国际收支平衡

### 2. 货币政策工具
| 工具 | 操作 | 影响 |
|------|------|------|
| 降准 | 降低存款准备金率 | 增加市场流动性 |
| 降息 | 降低基准利率 | 降低融资成本 |
| 公开市场操作 | 买卖国债 | 调节基础货币 |
| 结构性工具 | 定向支持 | 引导资金流向 |

### 3. 财政政策
- **扩张性**：减税降费、增加支出
- **紧缩性**：加税、减少支出
- **跨周期调节**：平滑经济波动

### 4. 经济周期判断
| 阶段 | 特征 | 资产配置 |
|------|------|---------|
| 复苏 | GDP上行、通胀低 | 股票 |
| 繁荣 | GDP上行、通胀高 | 商品 |
| 滞胀 | GDP下行、通胀高 | 现金 |
| 衰退 | GDP下行、通胀低 | 债券 |

## 三、行业分析方法论

### 1. 商业模式分析
**核心问题**：
- 为哪类客户解决什么痛点？
- 如何创造、传递、获取价值？
- 盈利模式是否可持续？

**分析框架**：
- 价值主张
- 客户细分
- 渠道通路
- 收入来源
- 成本结构

### 2. 竞争格局分析
**五力模型**：
1. 供应商议价能力
2. 买家议价能力
3. 现有竞争者
4. 新进入者威胁
5. 替代品威胁

### 3. 护城河类型

| 类型 | 案例 | 特点 |
|------|------|------|
| **无形资产** | 茅台品牌、专利 | 难以复制 |
| **转换成本** | 腾讯生态、苹果 | 用户粘性高 |
| **网络效应** | 微信、淘宝 | 用户越多价值越大 |
| **成本优势** | 海康威视、规模经济 | 成本低于对手 |
| **有效规模** | 自然垄断 | 新进入者无利可图 |

### 4. 行业生命周期
| 阶段 | 特征 | 投资策略 |
|------|------|---------|
| 导入期 | 高增长、高亏损 | 风险投资 |
| 成长期 | 高增长、盈利改善 | 成长投资 |
| 成熟期 | 低增长、高盈利 | 价值投资 |
| 衰退期 | 负增长、盈利下滑 | 回避 |

## 四、财务报表分析

### 1. 三张报表的关系

| 报表 | 体现 | 核心问题 |
|------|------|---------|
| **资产负债表** | 实力 | 公司有多少家底？ |
| **利润表** | 能力 | 公司能赚多少钱？ |
| **现金流量表** | 活力 | 公司现金流健康吗？ |

### 2. 资产负债表分析要点

**资产端**：
- 货币资金：现金充裕度
- 应收账款：回款能力
- 存货：周转效率
- 固定资产：产能规模

**负债端**：
- 短期负债：偿债压力
- 长期负债：财务杠杆
- 负债率：财务风险

**所有者权益**：
- 净资产：股东权益
- 未分配利润：累积盈利

### 3. 利润表分析要点

**关键指标**：
- 毛利率 = (营收 - 成本) / 营收
- 净利率 = 净利润 / 营收
- ROE = 净利润 / 净资产
- 营收增长率：成长性
- 净利润增长率：盈利能力

### 4. 现金流量表分析要点

**三大活动**：
| 活动 | 正值含义 | 负值含义 |
|------|---------|---------|
| 经营活动 | 主业造血 | 主业失血 |
| 投资活动 | 处置资产 | 扩张投资 |
| 筹资活动 | 融资借钱 | 还债分红 |

**关键指标**：
- 经营现金流净额 > 净利润 → 盈利质量高
- 自由现金流 = 经营现金流 - 资本支出

## 五、估值方法体系

### 1. 相对估值法

| 方法 | 公式 | 适用场景 |
|------|------|---------|
| **PE** | 股价/EPS | 稳定盈利的成熟公司 |
| **PB** | 股价/BPS | 重资产行业、金融业 |
| **PS** | 市值/营收 | 亏损的成长公司 |
| **PEG** | PE/增长率 | 成长股估值 |
| **EV/EBITDA** | 企业价值/息税折旧前利润 | 资本密集型行业 |

### 2. 绝对估值法

**DCF模型（现金流折现）**：
```
企业价值 = Σ(未来现金流 / (1+折现率)^n) + 终值
```

**关键参数**：
- 自由现金流预测
- 折现率（WACC）
- 永续增长率

**适用场景**：现金流可预测的成熟企业

### 3. 估值方法选择

| 公司类型 | 首选方法 | 辅助方法 |
|---------|---------|---------|
| 成熟盈利企业 | PE、DCF | PB |
| 高成长企业 | PEG、PS | DCF |
| 重资产企业 | PB | PE |
| 周期性企业 | PB、EV/EBITDA | PE（周期底部） |
| 亏损企业 | PS | PEG |

### 4. 估值判断标准

| 指标 | 低估 | 合理 | 高估 |
|------|------|------|------|
| PE | <行业均值 | ≈行业均值 | >行业均值 |
| PB | <1 | 1-3 | >3 |
| PEG | <1 | 1-2 | >2 |
| 股息率 | >5% | 2-5% | <2% |

## 七、实战要点总结

### 买入前的灵魂三问
1. 这家公司靠什么赚钱？（商业模式）
2. 为什么它能持续赚钱？（护城河）
3. 当前价格是否便宜？（估值）

### 持仓管理的核心原则
1. 分散投资：不把鸡蛋放一个篮子
2. 动态调整：根据基本面变化调仓
3. 止损纪律：错了就认，不摊平亏损

### 卖出的三个理由
1. 基本面恶化：护城河被侵蚀
2. 估值过高：透支未来多年增长
3. 发现更好机会：机会成本考量
