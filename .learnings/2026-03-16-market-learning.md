# 二郎学习笔记 - 2026年3月16日

## 一、市场热点与主线

### 当前最强主线
1. **AI算力链** - CPO/光模块、AI服务器、液冷、半导体
   - 催化：英伟达GTC 2026聚焦Rubin新GPU、CPO光互联
   - 代表：中际旭创、新易盛、天孚通信、工业富联、浪潮信息

2. **能源/储能/电力** - 油气、煤炭、储能、绿电、特高压
   - 催化：中东局势推升油价（布油破100美元）、算电协同政策
   - 代表：华电能源、中国石油、中国神华、阳光电源

3. **风电设备** - 风电整机、塔筒、海缆
   - 催化：英国4月起取消33项风电组件进口关税
   - 代表：大金重工、金风科技、天顺风能、通裕重工

4. **化工/顺周期** - 基础化工、农化、煤化工
   - 催化：油价上行+春耕旺季+钢厂复产预期
   - 代表：赤天化、红宝丽、川金诺、金牛化工

### 资金高低切换特征
- 资金从高位科技、算力、IT服务等方向流出
- 转向低位、低估值、高景气的顺周期板块
- 安全边际更高、持续性更强

## 五、龙虎榜解读要点

### 资金类型识别
- **机构专用**：公募/社保/险资，看重题材逻辑
- **深股通/沪股通**：外资，双向交易
- **拉萨军团**：散户集中营，分歧大
- **知名游资席位**：章盟主、赵老哥、方新侠等

### 龙虎榜预警信号
- 买一深股通 + 买二至买五拉萨军团 = 外资分歧+散户抱团
- 机构大幅卖出 + 游资撤离 = 高位兑现
- 机构买入 + 游资共振 = 主升浪
