# 金融研报生成完整工作流

## 报告类型矩阵

| 类型 | 场景 | 核心模块 | 数据源 |
|------|------|----------|--------|
| 盘前速递 | 09:15前 | 竞价/高开板块/隔夜全球/今日看点 | fetch_market + 隔夜美股/商品 |
| 午间快报 | 11:30-13:00 | 半日行情/资金/板块/午后展望 | fetch_market + web_fetch新闻 |
| 收盘总结 | 15:00后 | 全日行情/涨跌停/龙虎榜/策略 | fetch_market + web_fetch新闻 |
| 行业专题 | 任意 | 行业链/竞争/估值/趋势/标的 | web_search + web_fetch |
| 大宗商品周报 | 周末 | 商品周线/宏观驱动/持仓/展望 | fetch_market + web_fetch |
| 宏观经济月报 | 月初 | 经济数据/政策/利率/汇率/展望 | web_search + fetch_market |

## 数据采集流程

### Step 1: 结构化行情
```bash
python3 scripts/fetch_market.py > /tmp/market_data.json
```
输出包含: A股10大指数、港股3大指数、美股3大指数、欧洲3大指数、8个商品期货、4个外汇对

### Step 2: 新闻快讯
优先级：
1. `web_fetch https://www.cls.cn/telegraph maxChars=15000` — 财联社电报（最佳）
2. `web_fetch https://finance.eastmoney.com/` — 东方财富（龙虎榜/融资数据）
3. `python3 scripts/fetch_news.py` — 备用采集
4. `web_search` — 补充特定话题

### Step 3: 行业/专题数据（按需）
- `web_search "行业关键词 2024"` — 搜索最新行业动态
- `web_fetch` — 抓取具体文章/报告
- 同花顺行业数据: `web_fetch https://q.10jqka.com.cn/thshy/`
- 申万行业分类: `web_fetch https://www.swsresearch.com/`

## HTML生成规范

### 设计原则
- 宽度固定 **1280px**（截图标准宽度）
- **深色主题**：背景 #0a0e1a，卡片 #111827
- **红涨绿跌**（中国市场习惯）
- 数字用 **等宽字体**（Menlo/Courier New）
- 中文用 **PingFang SC / Microsoft YaHei**
- ❌ 不用 Google Fonts（截图可能无网络）
- ❌ 不用 CSS 动画（headless截图捕获中间状态）
- ❌ 不用外部图片链接（用内联SVG或emoji替代）

### CSS变量体系
参考 `assets/template.html` 中的 `:root` 变量定义。核心配色：
- `--accent-red: #ff4757` — 涨/警告
- `--accent-green: #2ed573` — 跌/安全
- `--accent-gold: #ffa502` — 重点/策略
- `--accent-blue: #3742fa` — 信息
- `--accent-cyan: #00d2d3` — 实时标签

### 模块库（按需选用）

**必选模块**（所有报告类型）：
1. **Hero头部** — 标题/日期/状态标签/一句话摘要
2. **免责声明Footer** — 法律声明

**行情模块**（盘前/午间/收盘版）：
3. **A股指数卡片** — grid布局，显示价格/涨跌幅/涨跌额
4. **港股/美股/欧洲** — 全球市场联动
5. **商品期货条** — 黄金/白银/原油等grid-4布局
6. **外汇面板** — 人民币/美元指数

**分析模块**：
7. **板块热力图/排行** — 领涨/领跌板块TOP10
8. **涨跌停统计** — 涨停/跌停/连板统计表格
9. **主力资金流向** — 北向资金/融资融券
10. **龙虎榜** — 机构/游资席位

**资讯模块**：
11. **新闻快讯** — 带分类标签(政策/行业/公司/全球)的新闻条目
12. **事件追踪** — 重大事件时间线（如地缘冲突）
13. **紧急告警** — 市场剧烈波动时的醒目Banner

**策略模块**：
14. **策略观点** — 基于行情的分析总结和操作建议
15. **风险提示** — 当前市场风险点
16. **关注标的** — 推荐关注的行业/个股（仅做研究参考）

**专题模块**（行业研报/周报/月报）：
17. **行业产业链图** — 上下游关系（用HTML/CSS绘制）
18. **竞争格局** — 主要玩家对比表格
19. **估值分析** — PE/PB/PS历史分位
20. **趋势图表** — 时间序列折线（纯CSS或内联SVG）
21. **数据仪表盘** — 关键指标KPI卡片

### 布局类名


## 盘前 vs 收盘差异

| 维度 | 盘前版 | 收盘版 |
|------|--------|--------|
| 时间 | 09:15前 | 15:00后 |
| 标注 | "盘前速递" | "收盘总结" |
| 数据 | 竞价/隔夜 | 收盘价 |
| 板块 | 高开预测 | 实际涨跌 |
| 成交 | 无 | 两市成交额 |
| 策略 | 今日看点 | 明日建议 |
