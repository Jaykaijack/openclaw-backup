## 核心学习要点

### 情绪判断
- 退潮期≠冰点，需区分小票情绪和大盘情绪
- 利好出尽是利空
- 放量下跌需警惕（主力出货）
- **新增**: 冰点(10%上涨)次日往往反弹，需识别情绪拐点

### 超跌反弹规律
- 昨日跌幅最大的板块，今日往往反弹最快
- 航空股案例：昨日-6.47%→今日+3.47%
- 不要线性外推单日涨跌

### 板块轮动规律（3月25日新增）
- 主线分化时，资金快速轮动到其他板块
- 化工板块今日强势，可能是电力资金溢出效应
- **应用**: 主线分歧日，关注资金轮动方向

### 高位连板股分歧信号（3月25日新增）
- 高位连板股（8板以上）分歧风险加大
- 板块内跟风股率先调整
- **应用**: 高位股分歧时，减仓跟风股

### 资金流向反转（3月25日新增）
- 南向资金从净卖出270亿→净买入100亿
- 资金流向具有高波动性
- **应用**: 不宜线性外推单日资金数据

### 期货-股票联动（3月25日新增）
- 原油暴跌 → 化工成本降 → 化工股利好
- 原油暴跌 → 石油开采股承压
- **错误案例**: 今日原油暴跌-6%，我判断"化工回避"，实际化工领涨
- **应用**: 期货异动时，需分析对相关板块的影响

### 认知偏差识别（3月25日新增）
1. **锚定效应**: 过度锚定昨日主线，忽视市场变化
2. **确认偏差**: 只关注支持判断的证据
3. **近期偏差**: 过度重视昨日数据
4. **过度自信**: 连续两日50%后未及时调整

### 选股策略
- 龙头策略优于分散持仓
- 集中持仓2-3只核心标的
- 新能源主线明确
